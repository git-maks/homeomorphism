About Graph Homeomorphism Checker

Overview

An interactive web-based tool for visualizing and checking graph homeomorphism and isomorphism. This educational application helps users understand topological and structural equivalences in graph theory through hands-on experimentation.

Algorithms Explained

Homeomorphism Detection

Two graphs are homeomorphic if one can be transformed into the other by subdividing edges (inserting degree-2 nodes along edges). This represents a topological equivalence.

Algorithm Steps:

  • Step 1: Simplify both graphs by iteratively removing all degree-2 nodes
  • Step 2: When removing a degree-2 node, connect its two neighbors directly
  • Step 3: Check if the simplified graphs are isomorphic
  • Step 4: If simplified graphs match, the original graphs are homeomorphic

Complexity Analysis: O(nxb2 + m) for simplification process, where n is the number of nodes and m is the number of edges, plus the cost of isomorphism checking.

Isomorphism Detection

Two graphs are isomorphic if there exists a one-to-one mapping between vertices that preserves adjacency. This represents a structural equivalence.

Algorithm Steps:

  • Quick Rejection Tests: Compare node counts, edge counts, and degree sequences
  • Degree Sequence: Sort nodes by their degree and verify both graphs have identical sequences
  • Backtracking Search: Use recursive backtracking to find a valid vertex mapping
  • Adjacency Verification: For each candidate mapping, verify all edges are preserved

Complexity Analysis: O(n!) in worst case (NP-complete problem), but optimized with pruning techniques. Practically efficient for graphs up to ~30 nodes.

Graph Theory Concepts

Key Definitions

  • Graph Homeomorphism: A topological equivalence relation. Two graphs have the same "shape" when degree-2 nodes (edge subdivisions) are ignored.
  • Graph Isomorphism: A structural equivalence. Two graphs have identical connectivity patterns with a perfect node-to-node correspondence.
  • Degree-2 Node: A node with exactly two connections, representing a subdivision point along an edge.
  • Degree Sequence: A sorted list of all node degrees, used as a quick structural fingerprint.

Relationship Between Concepts

  • All isomorphic graphs are homeomorphic (isomorphism ‚Üí homeomorphism)
  • Homeomorphic graphs may have different numbers of nodes due to edge subdivisions
  • Isomorphic graphs must have exactly the same number of nodes and edges
  • Homeomorphism is a weaker equivalence relation than isomorphism

Usage Instructions

Creating Graphs

  • Add Node: Left-click on empty space in either graph panel
  • Remove Node: Right-click on any node to delete it
  • Create Edge: Click and drag from one node to another

Checking Equivalence

  • Homeomorphism: Checks if graphs have the same topological structure (allows subdivisions)
  • Isomorphism: Checks if graphs have identical structure (exact match)

Example Graphs

Use the dropdown menu to load pre-built graph pairs that demonstrate different homeomorphism and isomorphism scenarios.

Educational Value

This tool is designed for students and enthusiasts learning graph theory. It provides immediate visual feedback and helps develop intuition about topological vs. structural graph properties.

Note: While the algorithms are correct for educational purposes, production-grade graph analysis systems would use more sophisticated approaches for large-scale graphs (e.g., Weisfeiler-Lehman algorithm, canonical labeling).
